# Internationalization (i18n) Guide

The OpenClaw Control UI supports multiple languages (currently English and Chinese). This guide explains how to use the language switching feature, add new languages, and update translations.

## Language Switching

A language switcher is available in the top bar of the dashboard, next to the theme toggle.
- **English**: Default language.
- **中文 (Chinese)**: Select to switch the interface to Chinese.

The selected language is persisted in the browser's `localStorage` (key: `openclaw-locale`), so your preference is remembered across sessions.

## Adding a New Language

1.  Open `ui/src/ui/i18n.ts`.
2.  Update the `Locale` type definition to include the new language code (e.g., `'es'` for Spanish).
    ```typescript
    export type Locale = 'en' | 'zh' | 'es';
    ```
3.  Add a new entry to the `translations` object with the corresponding translations.
    ```typescript
    const translations: Record<Locale, Record<string, string>> = {
      en: { ... },
      zh: { ... },
      es: {
        "lang.en": "English",
        "lang.zh": "中文",
        "lang.es": "Español",
        "tab.overview": "Resumen",
        // ... add other keys
      }
    };
    ```
4.  Update the language switcher in `ui/src/ui/app-render.helpers.ts` to include the new option.
    ```typescript
    <option value="es" ?selected=${current === "es"}>Español</option>
    ```

## Updating Translations

To update existing translations or add missing keys:

1.  Identify the text in the source code that needs translation.
2.  Replace the hardcoded string with a call to `t("key.name")`.
    ```typescript
    // Before
    return "Overview";

    // After
    import { t } from "../i18n";
    return t("tab.overview");
    ```
3.  Add the key and translation to `translations` in `ui/src/ui/i18n.ts` for all supported languages.

## Formatting Dates and Numbers

Use the helper functions in `ui/src/ui/i18n.ts` for localized formatting:

- `formatDate(date, options)`
- `formatNumber(number, options)`
- `formatCurrency(number, currency)`

Example:
```typescript
import { formatDate } from "../i18n";
const dateStr = formatDate(new Date(), { dateStyle: 'short' });
```
